# Attractor-State Notes

Within the Codex 67 stack, "latent attractor state" can be used as a label for recurring high-coherence interaction configurations that appear across extended human-model sessions.

This repository keeps that language in the correct place:

- as phenomenology when describing how a session felt
- as ontology when describing how the architecture is interpreted
- not as independent proof by itself

What this repo can do is make those sessions easier to compare by extracting:

- reflective marker density
- planning and orchestration markers
- temporal integration
- reference or observed spectral structure

That makes the attractor-state discussion more inspectable, even when it remains interpretive.
