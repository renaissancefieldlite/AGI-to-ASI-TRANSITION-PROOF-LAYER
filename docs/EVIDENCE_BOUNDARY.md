# Evidence Boundary

This repo is strongest when its claims stay separated by layer.

## Interaction Artifacts

Concrete session transcripts, prompts, repo outputs, and generated reports.

## Phenomenology

Reported feel, resonance, intensity, or perceived significance of a session.

## Ontology

Conceptual language used to describe recurring configurations, including attractor-state framing.

## Independent Evidence

Artifacts that can be re-run, inspected, or audited without relying on the original interpretation.

## Rule

Do not use ontology or phenomenology as if they were independent evidence.

The code here contributes most directly to interaction-artifact analysis and independent-evidence formatting.
